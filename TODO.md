## New Suggestions

#### Progression Depth
- [ ] **Achievement system** — milestone rewards that persist across prestige: "Kill 500 enemies → +2% gold forever", "Clear Stage 50 → unlock new class skin". Shown in a dedicated panel.
- [ ] **Item crafting** — spend gold + skill shards to: reroll a random substat, upgrade itemLevel by 1, or fuse 3 same-rarity items into a guaranteed higher-rarity item.

#### Combat Strategy
- [x] **Elemental weakness system** — enemies have an element type (fire, frost, holy, shadow, nature). Heroes deal +25% damage when their class element counters the enemy's. Show weakness icon on enemy card.
- [x] **More enemy status effects** — enemies can now inflict: bleed (DoT), freeze (halved SPD for 2 turns), silence (blocks skills for 1 turn). Show on hero card like existing debuffs.
- [ ] **Boss mechanics** — bosses above stage 50 get unique phase abilities: enrage at 50% HP, summon minions at 25%, AoE wipe threat at 10%. Preview the mechanic in the stage info before entering.

#### Economy & Loot
- [ ] **Daily dungeon** — one special stage per day with 3 locked modifiers and 2× drop rate. Resets at midnight. Rewards a guaranteed rare+ item and bonus skill shards.
- [ ] **Rune socketing** — rare+ items gain 1–2 rune sockets. Runes are small stat chips (e.g. "+5 ATK", "+2% crit") that can be swapped freely. Runes drop from bosses and can be bought in the shop.
- [ ] **Codex / bestiary** — track every enemy type killed (count + first-kill date). Each entry unlocks a short lore blurb. Bosses reward a small permanent bonus on first kill (e.g. +1% DEF forever).

#### UI & Feel
- [x] **Damage breakdown chart** — after each stage victory, show a small bar chart of damage dealt per hero (and healing done). Visible in the post-combat summary panel.
- [ ] **Hero comparison** — in HeroDetailModal, add a "Compare" button to select a second hero and view both stat columns side by side.
- [x] **Stage speed records** — track fastest clear time per stage. Show a 🏆 badge on stage panel when you beat your record. Visible in the battle statistics modal.
- [x] **Animated idle poses** — hero cards subtly animate between fights (a gentle pulse or breathing effect on the class icon) to make the idle state feel alive.
